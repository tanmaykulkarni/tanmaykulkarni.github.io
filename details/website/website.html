I designed this site so you can get to know me on a personal level. My work as an android developer, my portfolio, my design thinking, and everything else. After having made a few projects I had a feeling that there should be a website which would showcase my work that I love doing so much.

To start off with I decided to give a simplistic yet liberal approach while designing the content layout of the web pages on the site. I implemented Materialize - a CSS framework, which is a modern responsive front-end framework based on Material Design. I also made the use of Font Awesome - a CSS toolkit, which is a suite of icons and fonts for websites.

This portfolio site of mine is also an evidence of my knowledge of HTML and CSS, along with UI/UX design. I decided to host the site via Github Pages but this may change in the future.
