Tech Stack



I designed this site so you can get to know me on a personal level. My work as an android developer, my portfolio, my design thinking, and everything else. I felt like nothing can be better than a website which would showcase my work that I love doing so much. A website which would be a visual proof of my expertise, taste, and personality.

To start off with I decided to give a simplistic yet liberal approach while designing the content layout of the web pages on the site. I implemented Materialize - a CSS framework, which is a modern responsive front-end framework based on Material Design. I also made the use of Font Awesome - a CSS toolkit, which is a suite of icons and fonts for websites. Inspired by Medium, this site also uses 'Reading Time' - a simple, lightweight jQuery plugin used to display an estimated time to read the text. This plugin is used across all the 'Read More' pages of this site. The '1 min read' you see above is the execution of this plugin.

This portfolio site of mine is also an evidence of my knowledge of HTML and CSS, along with UI/UX design. I decided to host the site via Github Pages but this may change in the future.

The Color RED -

This site consists of two colors - Alizarin Red and Crimson Red. Moreover, Crimson Red is my favorite color and very special to me. Physiologically Red means energy, passion, action, strength, and excitement. It is also a universal sign of danger and warning.

It is also the color to correlate with emotion because it's on both ends of the spectrum. On one end you have happiness, falling in love, infatuation with someone, passion, etc. On the other end, you've got the obsession, jealousy, danger, fear, anger, and frustration.

Besides this, RED is the color that stimulates the physical senses and gets things moving.
