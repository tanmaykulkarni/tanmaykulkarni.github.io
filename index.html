Hi, I'm Tanmay
and I am an Android Developer. a Front-end Developer. a UI/UX Designer. a Tech Enthusiast. a Free Thinker.

I'm an IT student, passionate about Design, Front End Development, Android Development and Technology in general. I'm a developer on a variety of technical and creative projects including games, personalization, and utilities that make life easy and fun.

What do I do ?

Android Development

This is my current field of interest. I professionally develop Android applications from the ground up and rely heavily on graphic design to make an aesthetically appealing and beautiful UI/UX. I have experience in making various types of android applications right from native to web based and also VR applications for Google Cardboard.

Web Development

iLabelit

iLabelit is a web application which was built in collaboration with MIT Media Labs during my internship at Srujana Center Of Innovation, Hyderabad. iLabelit consists of an annotation tool which helps in annotating retinal images. iLabelit was initially a desktop-first web application but along with the desktop-based web app, I also built a mobile based web app which would help in annotating retinal images on the smartphone.

StereoCam

View Details

StereoCam

StereoCam is a Virtual Reality android application that I developed in association with MIT Media Labs while interning at Srujana Innovation Center, Hyderabad. StereoCam offers stereoscopic camera views with a filter of fish-eye lens which helps the user to see distant objects closely. The primary objective of building this application was to test it on partially blind people. StereoCam was built as a part of a research project.

Pupil Tracker

View Details

Pupil Tracker

As the name says, Pupil Tracker is a hardware prototype developed to track the pupil of a human eye. I contributed to this project by building a Virtual Reality android application which is composed of a football image, moving in a specific pattern in stereoscopic views. This helps the user to keep attention on the moving image, while his pupil gets tracked in the background.

Roll The Dice

Roll the Dice is my maiden android application. I developed this application to get started with android development and as an individual app developer. Before there was Android, there were Samsung phones running BADA OS which was written in C++. One of my friends phones had a dice application on his BADA OS Samsung phone. Rethinking on that, I decided to make a similar version of the dice application by adding more useful functionalities. Thus, I started writing the code in Java to develop my first android application. Within 6 months of launch, the app was downloaded more than 10,000 times and was trending on the Google Play Store.
